# 🎨 Image Denoise & Sharpen Batch Tool

**图像批量降噪与清晰度增强工具**

这是一个用于批量图像处理的小工具：
只需要指定一个输入文件夹，程序就会自动对其中所有图片执行 **降噪 + 锐化** 操作，并将处理结果输出到新的目录中，非常适合清理扫描图片、提高素材清晰度、改善噪声问题等场景。


## ✨ 功能特点

### 🧹 1. 多种稀疏（降噪）模式

你可以通过 `config.yaml` 自由选择更适合你图片特性的降噪方式：

| 模式            | 说明    | 特点          |
| ------------- | ----- | ----------- |
| **gaussian**  | 高斯模糊  | 快速、基础降噪     |
| **mean**      | 均值滤波  | 简单、速度快      |
| **median**    | 中值滤波  | 对椒盐噪声非常有效   |
| **bilateral** | 双边滤波  | **保边降噪**，推荐 |
| **nlmeans**   | 非局部均值 | 效果最强，但较慢    |

> 不同需求只需改动配置文件即可，无需修改代码。

---

### 🔪 2. 多种锐化模式

降噪后会对图像进行二次强化，使边缘更清晰、视觉更锐利。

支持的锐化模式：

| 模式               | 说明     | 特点         |
| ---------------- | ------ | ---------- |
| **unsharp_mask** | 反锐化掩模  | 专业级锐化、推荐   |
| **laplacian**    | 拉普拉斯锐化 | 快速锐化       |
| **custom**       | 自定义锐化核 | 灵活调节，适合研究  |
| **adaptive**     | 自适应锐化  | 基于图像边缘智能强化 |

你可以自由开关锐化功能，也能调整参数强度。

---

### 📦 3. 全自动批量处理

运行程序后，它将：

1. 自动扫描输入文件夹的所有图片
2. 根据配置逐张执行降噪 + 锐化
3. 将结果保存到独立输出目录
4. 输出统计信息（成功数、失败数、总耗时等）

无需手动拖拽，一键完成批处理。

---

## ⚙️ 配置说明（config.yaml 可编辑）

你可以在 `config.yaml` 中自由修改：

```yaml
denoise:
  mode: bilateral     # 降噪方式
  strength: 10        # 自定义参数（视模式而定）

sharpen:
  enabled: true
  mode: unsharp_mask  # 锐化方式
  amount: 1.5         # 锐化强度
  radius: 2.0         # 反锐化参数
```

只需修改配置即可切换不同处理风格，非常灵活。

---

## 🚀 使用方式

1. 将要处理的图片放入 `input/` 目录
2. 配置 `config.yaml` 中的降噪/锐化模式
3. 运行程序
4. 在 `output/` 文件夹查看处理结果

无需额外操作，上手即用。
